Who Am I?

안녕하세요. 창의적인 웹퍼블리셔를 꿈꾸고 있는 3년차 디자이너 백은지입니다. 디자이너로 일하면서 홈페이지 제작에 관심을 갖게 되었고 웹퍼블리셔 과정 수업을 듣게 되었습니다. 도전하는 것을 좋아하며, 살아가는데 있어서 새로운것을 습득하는 과정을 중요시 여기는 저한테는 코딩이 굉장히 흥미롭게 다가왔습니다.

지금까지 많은것을 시도해왔지만 정말 나에게 맞는 일이라고 생각하게 된것은 코딩이 처음이었습니다. 처음 공부를 시작할때는 막막했지만 유튜브 강의를 들으면서 알게된 웹스토리보이님의 스터디를 들으면서 여러 사람들과 정보를 공유하면서 공부를 즐겁게 할수 있었고, 좋아하는 일을 새롭게 시작하고자 늦은 나이지만 웹퍼블리셔라는 직업에 도전하게 되었습니다.

                                                            videoId: id,    
                                                            playerVars: player.query

                                                        // 리사이즈 화면 회전시 플레이어 크기 다시 설정

                                                        var viewport = {}, frame = {}, modal = {};
       
                                                        viewport.width = $( window ).width();
                                                        viewport.height = $( window ).height();
       
                                                        frame.width = viewport.width;
                                                        frame.height = frame.width / 1.6; // 16 : 10
