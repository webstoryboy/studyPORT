web standard
 웹 표준이란 브라우저 종류 및 버전에 따른 기능 차이에 대하여 호환이 가능하도록 제시된 표준으로, 다른 기종 혹은 플랫폼에 따라 달리 구현되는 기술을 동일하게 구현하여 웹 페이지를 제작하는 기법을 의미합니다. 웹 표준은 웹 접근성 준수를 위한 핵심이기 때문에 중요합니다.
 웹 표준을 준수한 사이트를 직접 제작하면서 장애인, 노약자를 포함한 모든 사용자가 동일한 조건에서 웹 사이트를 이용하려면 어떻게 해야 하는지 고려하게 되었고 검색 엔진 최적화, 서버 공간 절약 등의 장점들에 대해서도 알게 되었습니다.

responsive
responsive
반응형 웹 디자인(responsive web design, RWD)이란 하나의 웹사이트에서 PC, 스마트폰, 태블릿 PC 등 접속하는 디스플레이의 종류에 따라 화면의 크기가 자동으로 변하도록 만든 웹페이지 접근 기법을 의미합니다.
어떤 종류의 디스플레이로 접속해도 불편함이 없도록 미디어쿼리를 이용해서 반응형 웹 사이트를 제작했습니다. 시맨틱 태그를 사용하여 문서의 구조를 체계적으로 표현했습니다. 또한, 이미지 슬라이드와 라이트 박스를 구현하기 위해 자바스크립트 라이브러리를 활용했습니다.

megabox
메가박스 사이트입니다. swiper 자바스크립트 라이브러리를 사용하여 영화 목록이 자동으로 한 칸씩 옆으로 이동하도록 했고, 간단한 아이콘은 SVG를 활용하여 나타냈습니다. 새로운 영화 소식은 input 태그와 select 태그를 이용하여 구현했습니다. 시간 메뉴를 클릭하면 자신이 원하는 시간대를 선택할 수 있도록 시간 목록이 나타납니다.
미디어쿼리를 활용해서 반응형 사이트로 제작했고, 시맨틱 태그를 사용하여 코딩했습니다. 구글의 웹 API를 사용하여 새로운 영화를 클릭했을 때 유튜브의 영상이 레이어 팝업으로 나타나도록 작업했습니다.
