안녕하세요, Web Publisher 조유정입니다.

저는 4년간 광고디자이너로 일했습니다. 화보집·광고이미지·카달로그 등을 만들었고, 브랜드 로고디자인·화장품 패키지디자인 등 작업했습니다. 삼성물산·한섬 등 대형 패션회사와 일하면서 트렌드를 가까이 했고, 커리어도 쌓으며 자부심을 가지고 일했습니다. 또한 여러 클라이언트들을 만나면서 대화와 소통의 중요성을 깨달았고, 이해심과 빠른 적응력도 길렀습니다. 워낙 업무량이 많고 클라이언트와 일정을 조율하는게 매번 빠듯했기에 야근도 잦았지만 데드라인에 맞춰 시간을 컨트롤하며 작업하는 걸 확실히 배웠습니다.

점차 광고시장이 좁아지면서 진로를 고민하다 코딩수업을 듣게 되었습니다. 처음엔 가벼운 마음으로 공부했지만 점차 코딩의 매력을 느끼게 되었습니다. 머릿속에 상상하는 이미지를 코드를 통해 구현하는 것이 흥미로웠고, 한줄 한줄 작성하며 바로 결과를 확인할 수 있는 것이 정직하고 순수하게 느껴졌습니다. 시각적으로 어려움이 있는 사람들을 위해 웹표준을 준수하며 작업을 한다는 점도 제 마음에 작은 울림을 주었습니다.

어느 순간, 코드를 작성하며 집중하고 몰두하는 저를 발견하게 되면서 배움으로 끝나기보다 더 발전시켜보고 싶어 웹디자이너로 이직을 했습니다. 이직한 회사에서는 주로 템플릿을 활용해 반응형 홈페이지와 모바일App의 코딩을 했습니다. 디자인업무 60% 코딩업무 40% 정도의 비중이었는데, 점차 디자인보다 코딩에 더 매료되었고 템플릿 베이스보다는 좀더 탄탄하게 코딩을 해보고 싶어 웹퍼블리셔로 진로를 결정했습니다.

저에겐 배움이 삶의 원동력이고 곧 발전이라 생각합니다. 항상 배움에 열려있고 발전에 목말라있습니다. 빠르게 변화하는 웹 시장에 맞춰 공부할 자신이 있습니다. 앞으로 회사에 실질적인 도움이 되는 웹퍼블리셔로 일하고 싶습니다.

    // Youtube API

    (function($){
        (function handleTrailer(){
            // selector
            var $selector = {
                body: $( "body" ),
                overlay: $( "#blackout" ),
                modal: $( "#trailerModal" ),
                showButton: $( "#showTrailer" ),
                hideButton: $( "#hideTrailer" ),
            };
            
            // player
            var player = {
                obj: null, // player object
                query : {
                    theme: "dark",
                    color: "white",
                    controls: 1, 
                    autoplay: 1,
                    enablejsapi: 1,
                    modestbranding: 0, // YouTube 로고 감춤
                    rel: 0,  // 관련 동영상 표시
                    showinfo: 0, // 제목, 업로더 감춤
                    iv_load_policy: 3 // effect 감춤
                },
                visible: false
            };
            
            // 보이기, 숨기기 버튼 활성화
            $selector.showButton.on( "click", showPlayer );
            $selector.hideButton.on( "click", hidePlayer );
            
            // YouTube API를 이용해 iframe을 생성
            function setPlayer( id ){
                player.obj = new YT.Player( "trailer", {
                    width: 480,
                    height: 282,
                    videoId: id,    
                    playerVars: player.query
                });
                
                // 처음 플레이어 크기 설정
                resizePlayer();
                
                // resize 화면 회전시 플레이어 크기 다시 설정
                $( window ).on( "resize orientationchange", function() {
                    resizePlayer();
                });
            }
            
            function resizePlayer() {
                var viewport = {}, frame = {}, modal = {};
                viewport.width = $( window ).width();
                viewport.height = $( window ).height();
                frame.width = viewport.width;
                frame.height = frame.width / 1.6; // 16 : 10
                modal.top = ( ( viewport.height - frame.height ) / 2 ) + "px";
                modal.left = "0px";
                $selector.modal.css( modal );
                player.obj.setSize( frame.width, frame.height );
            }

            // iframe 보이기
            function showPlayer() {
                if ( !player.obj ) {
                    setPlayer( $selector.showButton.data("youtube") );
                }
                $selector.body.addClass("modal_on");
                $selector.overlay.show();
                player.visible = true;
            };
            
            // iframe 감추기
            function hidePlayer (){
                player.obj.stopVideo();
                $selector.overlay.hide();
                $selector.body.removeClass("modal_on");
                player.visible = false;
            };
        })();
        
    })(jQuery);
