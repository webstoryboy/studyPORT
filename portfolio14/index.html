ANIMATION

Heart Animation

CSS만을 이용한 애니메이션입니다. 가상요소로 하트 모양을 만들고 scale과 rotate를 이용해 뛰는 모양의 하트를 나타냈습니다.

Square Animation

가상요소로 박스에 그림자를 주고 애니메이션으로 border와 rotate를 주로 이용하여 로딩중인 화면처럼 표현하였습니다.

Gsap Animation

lettering 플러그인을 이용하여 span으로 글자를 각각 분할한뒤 애니메이션을 주고, 웹 접근성을 위하여 aria태그를 사용하였습니다.

SVG Text Animation

CSS만을 이용하여 애니메이션을 주었습니다. stroke-dasharray는 path를 점선으로 만들어주는데 stroke-dashoffset로 시작간격을 주어 텍스트 애니메이션을 완성하였습니다.
